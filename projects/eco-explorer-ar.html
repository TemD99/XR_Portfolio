XR Visualization Project

Explosive View

Explosive View is an XR visualization project designed to separate and reveal internal components in a clear, interactive layout. It helps users inspect parts individually while still understanding how the full object relates as a system.

Focus

3D inspection

The project focuses on pulling assemblies apart visually so users can examine structure, hierarchy, and placement.

Outcome

Better clarity

Users can understand complex assemblies faster when internal parts are separated and presented in a readable spatial layout.
